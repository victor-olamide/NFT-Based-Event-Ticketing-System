## 1. Set Up Your Wallet

### Step 1: Install a Web3 Wallet

**Recommended Wallets:**
- **MetaMask**: Most popular, browser extension + mobile
- **Coinbase Wallet**: User-friendly, built-in Base support
- **Trust Wallet**: Mobile-first, supports Base network
- **WalletConnect**: Compatible with many wallets

### Step 2: Add Base Network

**For MetaMask:**
1. Open MetaMask
2. Click network dropdown (top center)
3. Click "Add Network" â†’ "Add a network manually"
4. Enter Base network details:
   - **Network Name**: Base
   - **RPC URL**: https://mainnet.base.org
   - **Chain ID**: 8453
   - **Currency Symbol**: ETH
   - **Block Explorer**: https://basescan.org
5. Click "Save"

### Step 3: Get ETH on Base

**Option 1: Bridge from Ethereum**
1. Visit [bridge.base.org](https://bridge.base.org)
2. Connect your wallet
3. Enter amount to bridge (minimum ~$20 recommended)
4. Confirm transaction and wait for completion

**Option 2: Buy Directly**
1. Use Coinbase (easiest for Base)
2. Buy ETH and select Base network
3. Send to your wallet address

**Option 3: Use DEX**
1. Buy ETH on mainnet
2. Use Uniswap or other DEX to swap
3. Bridge to Base network

### âœ… Wallet Setup Complete
- Base network added and selected
- ETH balance shows in wallet
- Wallet connected to Base network

## 3. Purchase Your Ticket

### Step 1: Select Your Event
1. Click on the event you want to attend
2. Review all event details carefully
3. Check date, time, and venue
4. Verify ticket price and fees

### Step 2: Choose Your Seat (if applicable)

**General Admission:**
- Seat number assigned automatically
- First-come, first-served basis
- No specific seat selection

**Reserved Seating:**
- Interactive seat map available
- Click to select preferred seat
- Different sections may have different prices
- Unavailable seats shown in gray

### Step 3: Review Purchase Details

**Purchase Summary:**
```
Event: Base Music Festival 2024
Date: July 15, 2024, 8:00 PM
Venue: Madison Square Garden
Seat: Section A, Row 5, Seat 12
Ticket Price: 0.1 ETH
Gas Fee: ~0.001 ETH
Total: ~0.101 ETH
```

### Step 4: Complete Purchase
1. Click "Buy Ticket"
2. Your wallet will open
3. Review transaction details:
   - Recipient address (contract)
   - Amount (ticket price)
   - Gas fee estimate
4. Click "Confirm" in wallet
5. Wait for transaction confirmation (usually 2-5 seconds)

### âœ… Purchase Successful
- Transaction confirmed on blockchain
- Ticket NFT appears in your wallet
- QR code generated for entry
- Confirmation email sent (if provided)

## 5. Prepare for the Event

### Step 1: Event Reminders

**1 Week Before:**
- Confirm event details haven't changed
- Check venue location and directions
- Plan transportation and timing
- Verify QR code is accessible

**1 Day Before:**
- Charge your phone fully
- Download offline maps to venue
- Check weather and dress accordingly
- Prepare backup QR code copies

**Day of Event:**
- Arrive early to avoid crowds
- Bring phone charger/power bank
- Have QR code ready
- Bring ID if required

### Step 2: Technical Preparation

**Phone Setup:**
- Ensure phone is charged
- Increase screen brightness
- Clean screen for clear scanning
- Test QR code visibility

**Backup Plans:**
- Print QR code on paper
- Save screenshot to multiple devices
- Share with companion attending
- Note ticket ID number

### Step 3: Venue Information

**Research Venue:**
- Address and directions
- Parking availability and cost
- Public transportation options
- Entry procedures and timing
- Prohibited items list
- Accessibility information

## 6. Enter the Venue

### Step 1: Arrival at Venue
1. Locate the correct entrance
2. Join the appropriate line
3. Have QR code ready on phone
4. Prepare ID if required

### Step 2: QR Code Scanning

**Scanning Process:**
1. Venue staff will have scanning device
2. Present your phone with QR code
3. Staff scans the code
4. System verifies ticket authenticity
5. Ticket marked as "used"
6. Entry granted or denied

**Tips for Successful Scanning:**
- Keep phone steady
- Ensure good lighting
- Clean screen beforehand
- Increase brightness if needed
- Have backup ready

### Step 3: Troubleshooting Entry Issues

**QR Code Won't Scan:**
- Try different angle/distance
- Increase screen brightness
- Clean phone screen
- Use backup QR code
- Show ticket ID to staff

**Ticket Shows as Used:**
- Verify you're at correct event
- Check if ticket was transferred
- Contact venue staff immediately
- Show purchase confirmation

**Technical Issues:**
- Use printed backup
- Show wallet NFT to staff
- Provide ticket ID number
- Contact event support

### âœ… Successful Entry
- QR code scanned successfully
- Ticket verified as authentic
- Entry granted to venue
- Enjoy your event!

## 7. Transfer or Resell Tickets

### When You Can Transfer

**Check Transfer Settings:**
- Event must allow transfers
- Ticket not yet used
- Within transfer time limits
- Recipient has compatible wallet

### Step 1: Transfer to Friend/Family

**Free Transfer:**
1. Go to "My Tickets"
2. Select ticket to transfer
3. Click "Transfer"
4. Enter recipient's wallet address
5. Confirm transaction
6. Pay gas fee (~$0.25-1.00)

**Recipient Receives:**
- NFT ticket in their wallet
- QR code access
- Full ticket ownership
- Entry rights

### Step 2: Sell on Marketplace

**List for Sale:**
1. Go to "My Tickets"
2. Select ticket to sell
3. Click "List for Sale"
4. Set price (within allowed limits)
5. Confirm listing transaction

**Price Limits:**
- Maximum set by event organizer
- Usually 20-50% above original price
- Prevents excessive scalping
- Protects other fans

**Sale Process:**
```
Original Price: 0.1 ETH
Your Listing: 0.12 ETH (20% markup)
Platform Fee: 0.003 ETH (2.5%)
Organizer Royalty: 0.006 ETH (5%)
You Receive: 0.111 ETH
```

### Step 3: Buy from Marketplace

**Finding Tickets:**
1. Browse marketplace section
2. Filter by event/date/price
3. Compare available options
4. Check seller reputation

**Purchase Process:**
1. Select ticket to buy
2. Review all details
3. Confirm price and fees
4. Complete purchase
5. Ticket transfers to your wallet

- Show ticket ID to staff

- Check if transfers are allowed
- Ensure ticket not already used
