# Quick Start Tutorial

Get up and running with NFT Event Ticketing in 15 minutes or less.

## ‚ö° 5-Minute Setup

### What You Need
- Smartphone or computer
- 10 minutes of time
- $20+ worth of ETH (for tickets + fees)

### Step 1: Get a Wallet (2 minutes)

**Option A: Mobile (Recommended for beginners)**
1. Download **Coinbase Wallet** app
2. Tap "Create new wallet"
3. Choose username
4. Enable cloud backup
5. Set up Face ID/Touch ID

**Option B: Desktop**
1. Install **MetaMask** browser extension
2. Create new wallet
3. Set strong password
4. **CRITICAL**: Write down 12-word seed phrase
5. Store seed phrase safely

### Step 2: Add Base Network (1 minute)

**Coinbase Wallet:**
- Settings ‚Üí Active Networks ‚Üí Enable "Base"

**MetaMask:**
- Network dropdown ‚Üí Add Network ‚Üí Manual
- Name: Base
- RPC: https://mainnet.base.org
- Chain ID: 8453
- Currency: ETH

### Step 3: Get ETH on Base (2 minutes)

**Easiest Method:**
1. Go to [bridge.base.org](https://bridge.base.org)
2. Connect wallet
3. Bridge $50 worth of ETH from Ethereum
4. Wait 5-10 minutes for completion

**Alternative:**
- Buy ETH directly on Coinbase
- Send to Base network
- Transfer to your wallet

### ‚úÖ Setup Complete!
You're ready to buy tickets and use the platform.

## üé´ Buy Your First Ticket (5 minutes)

### Step 1: Connect to Platform
1. Visit NFT Event Ticketing website
2. Click "Connect Wallet"
3. Select your wallet
4. Approve connection
5. Ensure "Base" network is selected

### Step 2: Find an Event
1. Browse available events
2. Use filters (date, price, location)
3. Click on event you want to attend
4. Review all event details

### Step 3: Purchase Ticket
1. Select seat (if applicable)
2. Review purchase details:
   - Event name and date
   - Seat number
   - Total cost (ticket + gas)
3. Click "Buy Ticket"
4. Confirm transaction in wallet
5. Wait for confirmation (2-5 seconds)

### Step 4: Save Your Ticket
1. Go to "My Tickets" section
2. Find your new ticket
3. Screenshot the QR code
4. Save to phone gallery
5. Email QR code to yourself

### ‚úÖ Ticket Purchased!
You now own an NFT ticket and can enter the event.

## üèüÔ∏è Use Your Ticket (Event Day)

### Before Leaving Home
- [ ] Phone charged to 100%
- [ ] QR code screenshot saved
- [ ] Backup QR code emailed to yourself
- [ ] Event details confirmed (date, time, venue)
- [ ] Directions to venue planned

### At the Venue
1. **Locate Entry**: Find correct entrance for your ticket
2. **Prepare QR Code**: Open ticket in wallet or use screenshot
3. **Increase Brightness**: Make QR code clearly visible
4. **Present to Staff**: Venue staff will scan your QR code
5. **Enter Event**: QR code verifies instantly, enjoy the event!

### If QR Code Won't Scan
1. Clean phone screen
2. Increase brightness to maximum
3. Try different angle/distance
4. Use backup QR code
5. Show ticket ID to venue staff

## üîÑ Sell Your Ticket (If Plans Change)

### Check If Selling Is Allowed
- Event must permit transfers
- Ticket must not be used yet
- Must be within time limits

### List for Sale
1. Go to "My Tickets"
2. Select ticket to sell
3. Click "List for Sale"
4. Set price (within organizer limits)
5. Confirm listing transaction
6. Pay small gas fee (~$1)

### When It Sells
- Payment automatically sent to your wallet
- Ticket transfers to buyer
- You earn: Sale price - platform fee (2.5%) - royalty - gas

## üí° Pro Tips for Beginners

### Wallet Security
- **Never share your seed phrase** with anyone
- **Write seed phrase on paper**, don't store digitally
- **Use strong passwords** and enable biometrics
- **Keep wallet app updated** for security patches

### Saving Money
- **Buy tickets early** for best prices
- **Wait for low gas fees** during off-peak hours
- **Bridge larger amounts** to reduce bridge fees per transaction
- **Use "Standard" gas** instead of "Fast" when not urgent

### Event Day Success
- **Arrive early** to avoid entry crowds
- **Bring phone charger** or power bank
- **Have backup QR codes** in multiple places
- **Know venue policies** about prohibited items

### Marketplace Strategy
- **Research prices** before buying/selling
- **List early** for better visibility
- **Price fairly** within organizer limits
- **Be patient** for the right buyer/seller

## üö® Emergency Situations

### Lost Phone on Event Day
1. **Use backup device** with wallet app
2. **Check email** for QR code screenshot
3. **Ask friend** if you shared QR code
4. **Contact venue** with purchase confirmation
5. **Show ID** and explain situation

### Wallet Hacked
1. **Stop using wallet immediately**
2. **Create new wallet** on different device
3. **Transfer remaining funds** to new wallet
4. **Report to wallet provider**
5. **Change all related passwords**

### Transaction Stuck
1. **Check Base network status** at status.base.org
2. **Wait 10-15 minutes** for auto-resolution
3. **Speed up transaction** in wallet if available
4. **Contact support** if urgent

### Can't Access Tickets
1. **Check correct network** (should be Base)
2. **Refresh browser** and reconnect wallet
3. **Try different browser** or device
4. **Use wallet's built-in browser**
5. **Contact support** with transaction hash
