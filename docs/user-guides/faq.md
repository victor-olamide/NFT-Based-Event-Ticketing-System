# Frequently Asked Questions (FAQ)

Find answers to the most common questions about the NFT Event Ticketing System.

## üé´ General Questions

### What are NFT tickets?

NFT tickets are digital tickets stored on the blockchain as Non-Fungible Tokens (NFTs). Each ticket is unique, verifiable, and cannot be counterfeited or duplicated. They provide secure, fraud-proof event entry while enabling features like resale tracking and royalty payments.

### Why use NFT tickets instead of traditional tickets?

**Benefits include:**
- **Fraud Prevention**: Impossible to counterfeit or duplicate
- **Transparent Pricing**: All transactions visible on blockchain
- **Anti-Scalping**: Price limits and transfer controls
- **Instant Verification**: Real-time authenticity checking
- **Collectible Value**: Tickets remain as digital collectibles
- **Creator Support**: Royalties on secondary sales

### What blockchain network is used?

The system runs on **Base**, an Ethereum Layer 2 network that offers:
- Very low transaction fees (typically $0.25-2.00)
- Fast transaction confirmation (2-5 seconds)
- High security and reliability
- Easy integration with popular wallets

### Do I need cryptocurrency experience?

Basic knowledge is helpful but not required. Our guides cover:
- Setting up a Web3 wallet
- Getting ETH on Base network
- Making your first transaction
- Understanding gas fees
- Managing your tickets

## üí∞ Costs and Fees

### How much do transactions cost?

**Typical Costs on Base Network:**
- Creating an event: $1-3 in gas fees
- Buying a ticket: $0.25-1.00 in gas fees + ticket price
- Listing for resale: $0.50-2.00 in gas fees
- Marketplace purchase: 2.5% platform fee + royalties + gas

### What are gas fees?

Gas fees are network transaction costs paid to blockchain validators. On Base, these are very low compared to Ethereum mainnet. Fees vary based on network congestion but are typically under $2.

### Are there platform fees?

**Platform Fees:**
- **Primary Sales**: No platform fee (only gas fees)
- **Marketplace Sales**: 2.5% of sale price
- **Royalties**: 0-10% set by event organizers

### Can I get refunds?

Refund policies depend on the event organizer. Blockchain transactions are irreversible, so refunds must be processed manually by organizers. Check the event's refund policy before purchasing.

## üé™ Event Organizer Questions

### How do I create an event?

1. Connect your Web3 wallet
2. Click "Create Event"
3. Fill in event details (name, date, venue, etc.)
4. Set ticket parameters (price, quantity, transfer rules)
5. Configure anti-scalping protection
6. Set royalty percentage (optional)
7. Submit transaction and pay gas fee

### Can I change event details after creation?

Most event details cannot be changed after creation due to blockchain immutability. Plan carefully and double-check all information before submitting. Some platforms may allow limited updates through additional transactions.

### How do royalties work?

When you create an event, you can set a royalty percentage (0-10%). You'll earn this percentage on every resale of your tickets on the marketplace. Royalties are automatically distributed when tickets are sold.

### What anti-scalping options are available?

**Anti-Scalping Features:**
- **Maximum resale price**: Set price ceiling for resales
- **Transfer restrictions**: Disable resales entirely
- **Time limits**: Restrict transfers until certain date
- **Royalty fees**: Earn from resales to offset scalping

### How do I verify tickets at my venue?

Use the venue verification app to scan QR codes. The system instantly checks blockchain authenticity and marks tickets as used. See our [Venue Staff Tutorial](../tutorials/venue-staff-tutorial.md) for detailed instructions.

### How do I buy tickets?

Transfer ability depends on event settings:

3. Click "List for Sale"
4. Set price (within organizer limits)

6. Ticket appears on marketplace

3. Try different angle/distance
4. Use backup QR code copy
5. Show ticket ID to venue staff

## üõ°Ô∏è Security Questions

### Are NFT tickets secure?

Yes, extremely secure:
- **Blockchain verification**: Impossible to fake
- **Cryptographic signatures**: Each ticket mathematically verified
- **Immutable records**: Cannot be altered or deleted
- **Real-time validation**: Instant authenticity checking

### What if someone screenshots my QR code?

Screenshots won't work for fraudulent entry because:
- Each scan marks the ticket as "used"
- Only the first scan grants entry
- Blockchain prevents double-spending
- Venue staff can verify original ownership

### How do I keep my wallet secure?

**Security Best Practices:**
- Never share your private keys or seed phrase
- Use hardware wallets for large amounts
- Keep wallet software updated
- Verify website URLs before connecting
- Use strong passwords and 2FA when available

### What if I lose access to my wallet?

**Prevention:**
- Backup your seed phrase securely
- Store backup in multiple safe locations
- Consider hardware wallet for valuable tickets
- Test recovery process with small amounts

**If Lost:**
- Use seed phrase to recover wallet
- Contact wallet support for assistance
- Tickets remain on blockchain (not lost)
- May need technical help to recover access

## üîÑ Marketplace Questions

### How does the marketplace work?

The marketplace allows ticket holders to resell tickets before events. All transactions are secured by smart contracts with automatic escrow, preventing fraud and ensuring fair trades.

### What are the marketplace fees?

**Fee Structure:**
- **Platform fee**: 2.5% of sale price
- **Organizer royalty**: 0-10% (set by event creator)
- **Gas fees**: ~$0.25-2.00 for transactions
- **Total fees**: Typically 5-15% depending on royalty

### Can I negotiate prices?

The marketplace uses fixed-price listings. Sellers set their price, and buyers can purchase at that price. No auction or negotiation features are currently available.

### How do I know if a marketplace ticket is legitimate?

All marketplace tickets are automatically verified:
- **Blockchain authentication**: Impossible to list fake tickets
- **Smart contract escrow**: Payments protected until transfer
- **Transaction history**: Full ownership chain visible
- **Platform verification**: Additional security checks

## üÜò Troubleshooting

### Transaction failed - what do I do?

**Common Causes & Solutions:**
- **Insufficient gas**: Increase gas limit or wait for lower fees
- **Network congestion**: Try again later or increase gas price
- **Wallet connection**: Disconnect and reconnect wallet
- **Wrong network**: Ensure you're on Base network

### App won't connect to my wallet

**Troubleshooting Steps:**
1. Refresh the page
2. Clear browser cache
3. Try different browser
4. Update wallet extension
5. Restart browser/device

### QR code is not displaying

**Possible Solutions:**
- Check wallet NFT section
- Refresh platform page
- Try different device/browser
- Contact support with ticket ID
- Use backup QR code if available

### Marketplace listing not appearing

**Check These Items:**
- Transaction confirmed on blockchain
- Correct event selected
- Price within allowed limits
- Ticket not already sold/transferred
- Platform synchronization (may take few minutes)
