# Getting Started Guide

Welcome to the NFT Event Ticketing System! This guide will help you get started with creating events, purchasing tickets, and using the marketplace.

## üöÄ Quick Start

### What You Need

1. **Web3 Wallet** (MetaMask, Coinbase Wallet, or WalletConnect compatible)
2. **Base Network** added to your wallet
3. **ETH on Base** for transactions
4. **Internet Connection**

### Setting Up Your Wallet

#### Adding Base Network to MetaMask

1. Open MetaMask
2. Click the network dropdown (usually shows "Ethereum Mainnet")
3. Click "Add Network" ‚Üí "Add a network manually"
4. Enter these details:
   - **Network Name**: Base
   - **RPC URL**: https://mainnet.base.org
   - **Chain ID**: 8453
   - **Currency Symbol**: ETH
   - **Block Explorer**: https://basescan.org

#### Getting ETH on Base

**Option 1: Bridge from Ethereum**
1. Visit [bridge.base.org](https://bridge.base.org)
2. Connect your wallet
3. Enter amount to bridge
4. Confirm transaction

**Option 2: Buy directly on Base**
1. Use Coinbase (if you have account)
2. Purchase ETH and send to Base network
3. Or use other exchanges that support Base

## üí∞ Cost Structure

### Transaction Fees
- **Network Fees**: Base network gas fees (very low)
- **Platform Fee**: 2.5% on marketplace sales
- **Royalty Fees**: Set by event organizers (0-10%)

### Typical Costs
- **Creating Event**: ~$0.50-2.00 in gas
- **Buying Ticket**: ~$0.25-1.00 in gas + ticket price
- **Marketplace Sale**: 2.5% platform fee + royalties + gas

## üé´ Ticket Lifecycle

### 1. Event Creation
Organizer creates event with:
- Event details (name, date, venue)
- Ticket parameters (price, quantity)
- Transfer settings
- Royalty configuration

### 2. Ticket Sales
Buyers purchase tickets:
- Pay ticket price + gas fees
- Receive NFT ticket with QR code
- Ticket stored in wallet

### 3. Ticket Usage
At the event:
- Present QR code at venue
- Staff scans for verification
- Ticket marked as "used"
- Entry granted

### 4. Secondary Market (Optional)
Before the event:
- List ticket on marketplace
- Set sale price (within limits)
- Buyer purchases with automatic transfer
- Royalties paid to organizer
