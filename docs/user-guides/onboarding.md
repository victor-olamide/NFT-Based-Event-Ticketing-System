### Phase 3: Scanning Process (5 minutes)

**Basic Scanning**
```
Simple 4-step process:
1. Ask guest to present QR code
2. Position device 6-12 inches away
3. Wait for automatic scan (1-3 seconds)
4. Respond to result:
   üü¢ Green = Allow entry
   üî¥ Red = Deny entry
   üü° Yellow = Check with supervisor

That's it! The system handles all the complex verification.
```

**Handling Results**
```
Green (Valid):
- "Welcome! Enjoy the event!"
- Direct to appropriate entrance/section
- Move to next guest

Red (Invalid):
- "I'm sorry, this ticket shows as already used"
- "Let me get my supervisor to help"
- Remain polite and professional

Yellow (Warning):
- "One moment please, let me verify this"
- Try scanning again
- Call supervisor if needed

Response Guide: [Venue Staff Tutorial](venue-staff-tutorial.md#handling-verification-results)
```

### Phase 4: Troubleshooting (5 minutes)

**Common Issues**
```

- Ask guest to increase screen brightness
- Try different angle/distance
- Clean phone screen
- Use device flashlight if dark

Network Problems:
- App works offline for most scans
- Use backup verification methods
- Contact technical support
- Document manual entries

Guest Issues:
- Lost phone: Check email backups
- Ticket transferred: Verify with ID
- System error: Use manual verification

Troubleshooting Guide: [Venue Staff Tutorial](venue-staff-tutorial.md#troubleshooting-common-issues)
```

**Emergency Procedures**
