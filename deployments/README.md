# Deployment Information

This directory contains deployment information for the NFT Event Ticketing System on various networks.

## Base Network Optimization

The contracts are optimized for Base network with:
- Gas-efficient event creation with transfer restrictions
- Batch operations for reduced transaction costs
- Optimized parameter validation
- Base L2 specific gas estimation

## Files

- `base.json` - Base mainnet deployment info
- `base-sepolia.json` - Base Sepolia testnet deployment info
- `localhost.json` - Local development deployment info

Each deployment file contains:
- Contract addresses
- Transaction hashes
- Gas usage information
- Deployment timestamp
- Network configuration
